# Code of Conduct

## Purpose

This project is an honours dissertation for Edinburgh Napier University. While this is primarily a solo academic project, i welcome feedback, questions, and contributions from anyone interested in data cleaning, transparency, or privacy-first software.

## Expected Behavior

If you interact with this project (via issues, pull requests, or email), please:

- Be respectful and constructive in your feedback
- Focus on the technical content rather than personal matters
- Respect that this is a student project with academic deadlines
- Give credit where it is due when referencing this work

## Unacceptable Behavior

The following will not be tolerated:

- Harassment or discrimination of any kind
- Trolling, insulting, or derogatory comments
- Publishing private information without permission
- Any conduct inappropriate in an academic or professional setting

## Reporting

If you experience or witness unacceptable behavior, please contact me at calumxkerr@gmail.com. All reports will be handled respectfully and confidentially.

## Scope

This code of conduct applies to all interactions related to this project, including GitHub issues, pull requests, and email correspondence.
