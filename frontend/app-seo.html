offline-first via webassembly

what it does

this is an offline-first data cleaning tool for csv files. it is designed to run in your browser using webassembly (wasm), so your data can stay on your device.

  • detect missing values
  • detect and remove duplicate rows
  • produce a cleaned output you can download

transparent by design

i focus on clear, repeatable steps rather than black-box cleaning. the source code is public, and the goal is that the same input gives the same output.

source code: github.

privacy and offline

when the app runs in offline mode (webassembly), your csv stays on your device. when offline mode is not available, the app can fall back to api mode to run the same cleaning logic on the server.
