plain tools

what it does

small, direct tools for cleaning messy files without turning the page into a dashboard.

open the app honours project
  • csv, json/xml, and text workflows
  • audit-friendly output where possible
  • files are processed in memory and discarded
csv

spreadsheet cleaning

duplicates and missing values

upload a csv, paste data, or try a sample dataset. the toolkit is built around common spreadsheet cleanup jobs: duplicate rows, missing values, and simple standardisation.

  • detect duplicate rows
  • handle common missing-value markers
  • preview cleaned output before download

plain review flow

the interface stays deliberately simple. it shows the cleaning step, the output preview, and the operations log without hiding the work behind a decorative interface.

  • sample datasets for quick testing
  • downloadable cleaned files
  • repeatable cleaning behaviour
formats

json, xml, and text

semi-structured files

json and xml workflows are kept separate from csv so the tool does not pretend every file is a spreadsheet. upload or paste the data and inspect the cleaned result.

  • json upload and paste support
  • xml upload and paste support
  • download cleaned output

text cleanup

the text workflow is for simple unstructured cleaning jobs. it follows the same plain pattern: input, clean, inspect, download.

  • text file upload
  • clipboard paste
  • cleaned text preview
design

privacy and transparency

privacy-first handling

files are sent to the server, processed in memory, and discarded straight after cleaning. they are not written to disk, logged, or shared with third parties.

auditable changes

the project favours simple, inspectable algorithms over hidden magic. the aim is to make cleaning decisions easier to repeat and explain.
