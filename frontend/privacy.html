who we are

this website (tidy.calumkerr.com) and the data cleaning toolkit are operated by calum kerr as part of a final-year honours research project at edinburgh napier university. for any privacy questions, contact: calumxkerr@gmail.com.

for the purposes of uk gdpr, we are the data controller for any personal data processed through this site.

what data we process

the toolkit lets you clean data files (csv, json, xml, plain text) in your browser. two kinds of data are relevant:

1. the files you upload. when you upload a file and run a cleaning operation, the file contents are sent to our server over https, processed in memory, and returned to you. our application never writes your file contents to disk. as with any server, the operating system and the web server in front of the application may briefly buffer request data as part of normal operation; we configure the server to minimise this and such buffers are not retained. the contents of your file are never logged, never stored, and never shared with any third party. once your cleaning operation completes, the file contents are discarded from server memory.

if your file contains personal data (such as names, email addresses, postcodes, or anything else identifying), that data is briefly processed by the server but is not retained. the contents of your file are discarded immediately after processing.

2. technical request information. like any web server, ours records basic information about each request to help with security, debugging, and abuse prevention. specifically:

  • the ip address of the requesting device
  • the http method and path of the request (e.g. "post /api/clean")
  • the response status code
  • the timestamp

these records do not contain the contents of any file you upload. they are stored in server log files for up to 30 days, then automatically deleted.

what we do not do

to be explicit about things commonly done by other sites that we do not do:

  • we do not use cookies or browser local storage for tracking. the site does use a service worker to cache static assets so the app loads and stays usable offline, but no user data is stored client-side beyond the file you have currently loaded in your browser tab.
  • we do not use google analytics, plausible, hotjar, or any other analytics or telemetry service.
  • we do not load fonts, scripts, or other resources from third-party servers. all resources are served from our own domain.
  • we do not have user accounts, logins, or any kind of authentication. we do not collect names, email addresses, or any other contact information from users of the toolkit.
  • we do not share, sell, or transfer any data to any third party.
  • we do not send error reports, crash reports, or telemetry anywhere.

legal basis for processing

under uk gdpr, we rely on legitimate interests as our legal basis for processing technical request information (ip addresses, timestamps, paths). the legitimate interest is the operation, security, and improvement of a free public service.

we do not store the contents of files you upload, so no legal basis is needed for that — the data is processed only briefly to fulfil your explicit request to clean the file.

how long we keep data

  • file contents you upload: discarded immediately after processing (seconds, not stored).
  • technical request logs (ips, paths, timestamps): retained for up to 30 days, then automatically deleted by a scheduled task.

where data is processed

our server is hosted with ionos se, in a data centre in london, united kingdom. all data is processed within the united kingdom.

your rights

under uk gdpr, you have the following rights:

  • the right to be informed (this notice fulfils part of that).
  • the right of access to any personal data we hold about you.
  • the right to rectification if any of it is inaccurate.
  • the right to erasure ("the right to be forgotten").
  • the right to restrict processing.
  • the right to data portability.
  • the right to object to processing.
  • the right to lodge a complaint with the information commissioner's office (ico.org.uk).

to exercise any of these rights, email calumxkerr@gmail.com. we will respond within one month.

children

this site is not intended for children under 13. we do not knowingly collect data from children. if you are under 13, please do not use this site.

research and academic use

this toolkit is the deployed artefact of a final-year honours research project. the project itself has been examined and is no longer in active assessment. information about the research is available at honours-project.

a conference paper based on this work is in preparation. no personal data from toolkit users is used in the research or the paper — the research is about the toolkit's design and architecture, not about its users.

changes to this policy

if we materially change this policy, we will update the "last updated" date at the top. we recommend reviewing this policy periodically.
