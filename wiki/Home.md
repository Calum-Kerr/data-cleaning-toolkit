# Data Cleaning Toolkit

Welcome to the Data Cleaning Toolkit wiki.

## About

This is a privacy-first, transparent data cleaning toolkit built in C++ with WebAssembly support for offline use. It is developed as an honours dissertation project for Edinburgh Napier University.

## Key Features

- **Transparent Algorithms**: All cleaning algorithms are implemented from scratch and available for inspection
- **Offline Mode**: Runs entirely in your browser via WebAssembly - data never leaves your device
- **Online Mode**: Heroku backend API for server-side processing
- **Hybrid Fallback**: Tries WASM first, falls back to API if needed
- **No Coding Required**: Simple web interface for uploading and cleaning CSV files
- **Reproducible Results**: Same input always produces same output
